

## For client folder
- `npm i`
download node_modules
- `yarn start` 
for locally develop
- `yarn build`
generate static pages for deploy

## For server folder
- `npm start`
run up the server


## For docker
`docker-compose build`

in a terminal
- `docker-compose exec {service name} sh`
